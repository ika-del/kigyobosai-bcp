人命安全・初動体制の確立
地震発生直後に機能する「生命を守る仕組み」を最初に整備する。多くが法定義務。
即日〜1週間以内 0/6 完了

予算承認のみ（実務は総務部主導）

✓
緊急避難経路・避難場所の確認と掲示更新
各フロアの避難経路図・非常口・集合場所を更新し全従業員に周知。年1回以上の見直しが必要。
📌 消防計画の届出義務あり。未整備は消防法違反として指摘対象。
📋 実施担当：総務部／作業工数：半日〜1日／消防署への届出を忘れずに
コスト：低 総務部主導 消防法
✓
安否確認システムの導入と訓練実施
安否確認サービス（安否確認くん・YellowFin等）を導入し、全従業員登録後に訓練を実施。輻輳時でも機能する仕組みを確保する。
📌 発災後72時間以内の全員安否確認が顧客・取引先への説明義務に直結。
📋 比較推奨：安否確認くん（月額〜3万円）・TotalBCP（初期30万〜）。従業員スマホ登録率100%を必達目標に。
コスト：中 総務部主導
✓
緊急連絡網の整備・最新化（3系統確保）
役員・部門長・担当者の連絡先を携帯・固定・代替の3系統で管理。異動・退職のたびに更新する運用ルールを設ける。
📌 役員の緊急連絡先が不明確な事例が多発。役員分は経営層が直接確認・承認を。
📋 紙の連絡網も必ず作成・保管（停電・スマホ充電切れを想定）。更新担当者を明記。
コスト：低 総務部主導
✓
AED・救急用品の設置確認と使用期限管理
AED設置場所・使用期限を確認。救急箱を各フロアに配備し、全従業員に使用方法を周知する。
📌 AED設置・管理の不備は労働安全衛生法上の安全配慮義務違反につながる。
📋 AEDパッドの有効期限（機種により2〜4年）とバッテリー残量を年1回チェック。記録簿を作成。
コスト：低 総務部主導 労安法
✓
オフィス内の家具・什器の転倒防止措置
書棚・PC・サーバーラック等をL字金具・ベルトで固定。危険箇所リストを作成し優先順で実施。
📌 什器転倒による従業員の怪我は会社の安全配慮義務違反として損害賠償請求対象になる。
📋 施設管理会社に依頼する場合のコスト目安：50席規模で10〜20万円。実施前後の写真記録を必ず残す。
コスト：中 総務部主導
✓
帰宅困難者対応方針の策定と周知
発災時に帰宅できない従業員への対応ルール（施設滞在・食事・情報提供・宿泊）を文書化し社内掲示する。
📌 東京都条例では従業員の3日分の帰宅困難者対策が義務。違反は是正指導の対象。方針決定は経営判断が必要。
📋 対象人数×3日分の食料・水の備蓄数量を算出し購買部門に発注。フロア別の滞在スペースも事前に確保。
コスト：低 経営承認必要 東京都条例

フェーズ想定コスト

経営層の決定事項
防災責任者の選任・予算承認・方針表明
未対応リスク
消防法上の防火管理者未選任は罰則対象（30万円以下の罰金）
✓
防災責任者・防火管理者の選任（経営判断）
法令上の防火管理者（消防法）を選任し、BCPを推進する防災担当者を社内規程で任命する。役割と権限を文書化。
📌 防火管理者は延べ面積300㎡以上で法定義務。未選任は消防法違反・30万円以下の罰金。
→ 経営層が正式に選任・辞令発令する必要あり
📋 防火管理者講習（甲種：2日間、乙種：1日間）への受講手配。消防署への届出を3ヶ月以内に完了。
コスト：低 経営層主導 消防法 ★経営承認
✓
防災委員会の設置と経営層への定期報告体制
各部門代表を含む防災委員会を設置。四半期に1回、経営会議に進捗報告を行う仕組みを作る。
📌 防災投資は経営課題。取締役会への報告義務を設けることで、全社の本気度が変わる。
📋 委員会規程の作成・委員の選定・議事録管理を総務部が担当。初回会議は経営層が出席して方針を表明する。
コスト：低 経営承認必要 ★経営承認
✓
消防訓練の年2回実施計画と届出
避難訓練・安否確認訓練・初期消火訓練を年2回（春・秋）に計画。参加率90%以上を目標とし実施記録を保管。
📌 消防訓練の実施は法定義務。未実施は消防署の立入検査で是正指導の対象。
📋 訓練実施報告書を消防署に提出。外部講師（消防署・民間）の手配は2ヶ月前から。参加率を経営会議で報告する。
コスト：中 総務部主導 消防法
✓
全従業員への防災教育・研修の実施
新入社員研修への防災教育組み込み。年1回のeラーニングまたは全社防災セミナーを実施する。
📌 従業員の防災知識不足は「会社が教育義務を怠った」として安全配慮義務違反に問われうる。
📋 eラーニング教材（BizLearn等）の選定・契約が最も効率的。受講記録を保管し、法定訓練との区別を明確に。
コスト：中 総務部主導
✓
建物の耐震診断・補強工事の意思決定
旧耐震基準（1981年以前）の建物は耐震診断を実施。必要に応じて補強工事または移転を検討する。
📌 旧耐震基準の建物で人的被害が発生した場合、会社の使用者責任（民法415条）が問われる可能性がある。補強工事費用は助成金対応可能なケースあり（東京都など）。
→ 最終判断は経営層が行う
📋 建築士・耐震診断機関への診断依頼（費用：建物規模により20〜100万円）。診断結果を経営層に報告し判断を仰ぐ。
コスト：大 経営層主導 建築基準法 ★経営承認
✓
従業員の自宅防災状況の把握と支援制度検討
アンケートで自宅の耐震・備蓄状況を把握。備蓄品購入補助・耐震診断費補助などの福利厚生制度を検討する。
📌 自宅被災した従業員は出社困難になる。従業員の個人防災力向上は事業継続の土台。福利厚生として採用活動にも有効。
📋 アンケート設計・集計・分析を総務部が実施。支援制度の導入は人事部と連携し経営層に提案する。
コスト：低 総務部主導

PHASE 03 — 重要 ／ 予算承認が必要
備蓄・防災設備の整備
72時間（3日間）の自立を可能にする物資・設備を確保する。条例義務を含む。
1〜3ヶ月以内 0/6 完了
▼
経営インパクト
条例義務・従業員保護
フェーズ想定コスト
〜150万円
経営層の決定事項
予算承認・設備投資判断
未対応リスク
帰宅困難者条例違反・停電時の業務停止・従業員保護義務不履行
✓
非常食・飲料水の3日分備蓄（条例義務）
従業員1人あたり：飲料水3L×3日分・非常食3日分。帰宅困難者分を加算して在庫管理体制を構築する。
📌 東京都帰宅困難者対策条例で3日分の備蓄が義務。違反は公表・是正指導の対象。
📋 100人規模：飲料水900L（2Lペット×450本）＋食料300食分。保管スペース確保と年1回のローリングストック計画が必要。
コスト：中 総務部主導 東京都条例
✓
非常用電源・発電機の設置判断（経営投資）
72時間対応の発電機またはポータブル電源を確保。電力容量・燃料備蓄・設置場所を計画し導入を決定する。
📌 停電時に電話・PC・サーバーが止まれば顧客対応・経理処理が全停止。投資回収期間は「1回の停電被害コスト」で試算可能。
→ 設備投資判断は経営層の決裁が必要
📋 ガソリン発電機（30〜100万円）かポータブル電源（10〜50万円）を用途別に選定。燃料備蓄量の計算式：消費kW×時間÷燃費（L/h）。
コスト：高 経営層主導 ★経営承認
✓
防災備品セットの整備・フロア配備
ヘルメット・軍手・懐中電灯・防塵マスク・毛布・携帯トイレを人数分用意。部門別・フロア別に配備し場所を周知。
📌 発災直後に備品がなければ従業員の安全確保ができない。整備状況を役員巡回で確認することを推奨。
📋 100人規模の備品購入費目安：50〜80万円。防災備品専門業者（アスクル防災・ミドリ安全等）で一括調達が効率的。
コスト：中 総務部主導
✓
消火設備・スプリンクラーの法定点検実施
消火器の設置・有効期限確認、スプリンクラー・自動火災報知機の法定点検（年1回以上）を確実に実施する。
📌 消防設備の点検未実施・未報告は消防法違反。立入検査で指摘されると是正命令・公表の対象になる。
📋 消防設備点検業者（防火対象物点検資格者）と年間契約を結ぶことが最も確実。点検結果報告書を3年間保存。
コスト：中 総務部主導 消防法
✓
備蓄品の在庫管理・ローリングストック体制
食料・水の賞味期限を台帳で管理。ローリングストック方式で廃棄コストを最小化。年1回棚卸しを実施。
📌 期限切れ備蓄品を保管していても「備蓄あり」とは見なされない。訓練で実際に配布し消費する仕組みが有効。
📋 在庫管理台帳（Excelまたは備蓄管理アプリ）を作成し担当者を固定。9月の防災の日に合わせた棚卸しが定着しやすい。
コスト：低 総務部主導
✓
断水時対応トイレ・衛生環境の確保
携帯トイレを全員分（3日分×3回/日）確保。女性・障がい者への配慮を含む使用ルールを整備しておく。
📌 トイレが使えないと従業員は速やかに帰宅困難になる。衛生問題は二次被害にもつながる。
📋 携帯トイレ数の算出：従業員数×3日×3回。段ボールパーティション（1枚〜2万円）をセットで確保。保管場所も事前に確保。
コスト：中 総務部主導

情報・通信基盤の強化

経営層の決定事項
IT投資承認・情報発信ポリシー決定
未対応リスク

✓
衛星電話・MCA無線の調達（代替通信手段）
輻輳する携帯に依存しない通信手段を確保。本社・工場・拠点間で最低1台以上の衛星電話または無線機を調達。
📌 阪神大震災・東日本大震災でも携帯は数時間不通。衛星電話（イリジウム等）は月額〜1万円程度で維持可能。
→ 調達予算の承認は経営層が行う
📋 衛星電話レンタル（月額5,000〜2万円）と購入（30〜60万円）を比較検討。使用訓練を年1回実施し操作習熟を確保。
コスト：高 経営承認必要 ★経営承認
✓
業務データのクラウドバックアップ体制構築
重要データを毎日クラウド（AWS/Azure/Google等）に自動バックアップ。RTO・RPOを定義し復旧手順を文書化する。
📌 オフィス全焼・浸水でオンプレサーバーが消失した場合、クラウドバックアップがなければ業務データが永久に失われる。
📋 RTO（目標復旧時間）とRPO（目標復旧時点）を部門別に設定。IT部門が主管だが総務部が要件定義に参加する。月額コスト目安：数千円〜数万円。
コスト：中 総務/IT部
✓
テレワーク環境の整備と被災時稼働の検証
オフィス被災時に自宅・代替拠点から業務継続できるVPN・クラウド環境を整備。実際に試す訓練も実施する。
📌 テレワーク整備はBCPと平時の生産性向上を両立できる最も費用対効果の高い投資の一つ。
📋 全員がリモートで1週間業務継続できるかシミュレーション。VPN・クラウドツール（Microsoft365等）の契約見直しも同時に実施。
コスト：中 総務/IT部
✓
ラジオ・J-ALERTなど情報収集手段の整備
停電時も使える乾電池式ラジオ・手回しラジオを整備。防災気象情報アプリの一斉設定と情報担当者を指名する。
📌 正確な情報収集ができない状況での経営判断ミスは企業の信頼を大きく損なう。
📋 ラジオ（5,000〜1万円/台）をフロアごとに設置。Yahoo防災速報・NHKニュースアプリを全員インストール。情報担当は副総務部長が兼任する形が多い。
コスト：低 総務部主導
✓
顧客・取引先・金融機関への緊急連絡体制整備
主要顧客・仕入先・金融機関への緊急連絡フロー（担当者・連絡先・通知タイミング・内容テンプレート）を文書化する。
📌 取引先への連絡遅れは契約違反・損害賠償リスクになる。「何を・いつ・誰が・どう伝えるか」を事前に決めておくことが必須。
📋 主要取引先TOP20の緊急連絡先一覧を整備。連絡テンプレートを3パターン（軽微被害・中程度・重大被害）作成し金庫に保管。
コスト：低 経営承認必要
✓
公式SNS・ウェブサイトによる情報発信ポリシー策定
被災後の営業状況・復旧見込みをSNS・ホームページで発信する担当者・承認フロー・テンプレートを事前に準備する。
📌 被災後の不適切な情報発信は炎上・信用失墜のリスク。発信内容の最終承認者（代表者または指定役員）を事前に明確にしておく。
→ 発信ポリシーは経営層が決定する
📋 発信テンプレートを広報部門と共同作成。「被災直後の第一報→状況確認→復旧目途」の3段階の発信内容を準備。SNSアカウントの緊急時運用権限も確認。
コスト：低 経営層主導 ★経営承認
🔄
PHASE 05 — 経営主導 ／ BCP策定の核心
事業継続計画（BCP）の策定
「被災してもビジネスを止めない」ための計画を体系的に整備する。経営層の関与が最も重要なフェーズ。
6〜12ヶ月以内 0/6 完了
▼
経営インパクト
事業存続・廃業リスク
フェーズ想定コスト
〜200万円
経営層の決定事項
BCP方針決定・代替拠点契約・保険見直し・ステークホルダー開示
未対応リスク
被災後の廃業・取引先からの契約打ち切り・金融機関からの評価低下
✓
BIA（業務影響度分析）による重要業務の特定
全業務の中から「止められない重要業務」を経営レベルで選定。停止による財務・信用・法令上の影響を定量化する。
📌 「何を守れば会社が生き残るか」を経営層が決定しない限りBCPは絵に描いた餅になる。BIAは経営層主導で実施する。
📋 BIA調査票を部門ごとに配布・回収・集計し、経営会議で優先業務を決議する。中小企業庁の無料テンプレートを活用可能。
コスト：中 経営層主導 ★経営承認
✓
BCP文書の策定（内閣府ガイドライン準拠）
基本方針・被害想定・対応戦略・資源確保・連絡体制を網羅したBCP文書を作成。代表者の署名を付して正式文書化する。
📌 BCP文書は代表者名で発行することで全社への拘束力が生まれる。銀行・大手取引先がBCP有無を与信審査に含める事例が増加中。
📋 内閣府「事業継続ガイドライン」と中小企業庁「中小企業BCP策定運用指針」を参考文書として使用。総務部が事務局として草案作成→経営層が決裁。
コスト：中 経営承認必要 ★経営承認
✓
代替拠点・代替業務手段の確保（経営決断）
本社被災時の代替オフィス（コワーキング契約・グループ会社支援協定）を事前に確保する。
📌 本社が使えない状態で代替拠点の確保交渉を始めても遅い。有事前の契約・協定が生命線。
→ 契約・協定の締結は経営層が決裁する
📋 コワーキング候補リスト作成（WeWork・TKP会議室等）→経営層に提案→契約。交通アクセス・収容人数・設備を比較資料にまとめる。
コスト：高 経営層主導 ★経営承認
✓
サプライチェーンリスク評価と代替調達先確保
主要仕入先の被災リスクを評価し、代替サプライヤーを2社以上確保。集中調達リスクを分散する意思決定を行う。
📌 単一サプライヤーへの集中は取引先の被災で自社業務が即日停止するリスク。調達分散は経営戦略として決断する必要がある。
📋 主要仕入先TOP10を対象にリスク評価（立地・規模・BCP有無）を実施。代替サプライヤー候補リストを購買部門が作成し経営層に提案。
コスト：中 経営承認必要
✓
災害保険・地震保険の補償内容の見直し
財産保険・企業費用保険・利益保険の補償内容を確認。地震免責条項・支払い上限を精査し適切な補償へ見直す。
📌 多くの火災保険は「地震火災」を免責としている。地震後の損害は別途地震保険が必要。補償ギャップは経営層が把握・判断する。
→ 保険見直しの最終決定は経営層・財務責任者
📋 保険代理店に現行証券の補償内容レビューを依頼（無料）。地震リスク評価レポートを取り寄せ、費用対効果を経営層に報告する。
コスト：中 経営層主導 ★経営承認
✓
BCPのステークホルダー開示・取引先連携
顧客・投資家・金融機関に対してBCP策定状況を開示。主要取引先にもBCP連携（相互支援・情報共有）を働きかける。
📌 BCP開示は取引先・金融機関との信頼関係を強化する経営戦略。非財務情報開示の観点からもESG評価に直結する。代表者からの発信が最も効果的。
📋 開示資料は経営層が承認した後、総務部が取引先向けに配布。IRページへの掲載・営業担当者へのトークスクリプト提供も実施。
コスト：低 経営層主導 ★経営承認

✓
BCPの年次見直しと経営承認・再発行
組織変更・業務変化・法改正を反映して毎年BCPを更新。代表者の署名を付して再発行し全社に展開する。
📌 古いBCPは「意味がない」ではなく「有害」になりうる（誤った指揮系統・廃止された手順が記載されたまま）。経営層による年次承認が必須。
📋 年次改訂スケジュール：9月（防災月間）に改訂作業→10月経営会議で承認→11月全社展開。改訂履歴を文書に明記。
コスト：低 経営承認必要 ★経営承認
✓
BCP机上演習・実動訓練の年次実施（経営層参加）
年1回以上、具体的シナリオを設定したBCP演習を実施。経営層も参加し、実際の意思決定を体験する。
📌 経営層が訓練に参加しない会社は「いざとなったら役員が動けない会社」。経営層の参加が全社の本気度を決定的に変える。
📋 外部ファシリテーター（民間BCPコンサル）を使った机上演習が最も費用対効果が高い（1回30〜80万円）。演習後の改善アクションリストを作成し翌月に反映。
コスト：中 経営承認必要
✓
防災関連法令・ガイドライン改正の年次確認
消防法・建築基準法・労働安全衛生法・東京都条例等の改正を年1回確認し、必要な対応を実施する。
📌 法改正への対応漏れは行政指導・罰則リスクに直結。総務部からの年次報告を経営会議の議題に設定する。
📋 法令改正情報は消防署の定期通知・東京都ウェブサイト・防災コンサルからの情報を総務部が収集・整理して経営会議に報告。
コスト：低 総務部主導 各種法令
✓
地域・行政との連携強化（協定締結）
地元自治体・消防署・地区防災協議会との連携強化。帰宅困難者受け入れ協定・地域防災訓練への参加を進める。
📌 地域防災協力企業としての認定・表彰は採用ブランディング・行政との関係構築に有効。協定締結は代表者名で行う。
📋 地区防災協議会への参加申請→自治体との協定締結交渉。消防署主催の防災協力事業者登録も検討。総務部が窓口となり、締結時のみ経営層が対応。
コスト：低 経営承認必要
✓
防災投資の費用対効果評価と次年度予算決裁
当年度の防災対策コストと、リスク低減効果・保険見直し効果を比較評価し、次年度の防災予算を経営判断で決定する。
📌 防災投資を「感情論」ではなく「ROI」で評価することで経営会議での承認が通りやすくなる。費用対効果試算は総務部が資料作成→経営層が決裁。
📋 費用対効果試算の基本式：想定損害額×発生確率×リスク低減率 ÷ 投資コスト。数値の根拠は内閣府・東京都の被害想定データを使用。
コスト：低 経営層主導 ★経営承認
